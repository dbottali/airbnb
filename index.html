Audit e calcoli

Formula usata

Base lorda = accommodation + cleaning fee.
Payout prima ritenuta = base lorda - host service fee Airbnb.
Ritenuta Italia = base lorda × aliquota ritenuta.
Netto totale = payout prima ritenuta - ritenuta Italia.
Payout PM = netto totale × percentuale PM.
Payout Damiano = netto totale - payout PM.
Fondocassa = payout PM netto × cleaning fee / base lorda.
PM effettivo = payout PM netto - fondocassa.
